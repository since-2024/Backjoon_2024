## Dynamic Programming

- 특정 범위까지의 값을 구하기 위해 '이전 범위의 값을 활용'하여 효율적으로 값을 얻는 기법
- 이전 범위의 값을 저장(Memorization)하여, 똑같은 문제가 발생했을 때 이를 참조하여 해결함.
